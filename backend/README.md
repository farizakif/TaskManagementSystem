See the main README.md in the project root for complete documentation.

## Quick Start

1. Create PostgreSQL database: `CREATE DATABASE task_management_db;`
2. Update `application.properties` with your database credentials
3. Run: `mvn spring-boot:run`
4. Sample data will be automatically created on first run
